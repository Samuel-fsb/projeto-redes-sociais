📝 **Sobre o projeto**


Para aprender a utilizar iframe, posicionamento de imagens de fundo, centralização, navegação interna e outros conceitos de CSS3, construí este projeto que unifica todos esses elementos.

### Tecnologias Utilizadas
1. HTML5
2. CSS3
3. JavaScript
4. Git GitHub


### O que eu pratiquei?

Neste projeto, foquei em aplicar os seguintes conceitos:

* Utilização de iframes;
* Inserção de imagens no fundo via CSS;
* Navegação;
* Responsividade usando media query.
  

### Como funciona?
* O usuário pode navegar entre as seções do site e acessar as minhas redes sociais.
